## 微信小程序组件库

### 介绍
`Mind UI` 是使用微信小程序原生语言开发的轻量、高效的 `UI` 组件库。
目前包含 30+ 个常用组件。

### 注意
组件库目前还处于 `alpha` 阶段，不建议在生成环境使用。  
如果你在使用中有任何意见或建议，欢迎反馈。
